# Personal Portfolio Website

A premium, responsive personal engineering portfolio website for robotics, embedded systems, autonomous systems, research and embedded AI.

---

## 🚀 Features

- Modern single-page portfolio design
- Animated hero section
- Interactive cursor glow
- Scroll reveal animations
- Robotics-style terminal and radar visual
- Project showcase cards
- Experience timeline
- Leadership and social impact section
- Currently exploring section
- Responsive mobile navigation
- Dark engineering aesthetic
